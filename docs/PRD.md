# 產品需求文件（PRD）— 讀書筆記本系統

## 1. 產品概述

### 1.1 產品名稱
讀書筆記本（Book Notes）

### 1.2 系統描述
讓讀者可以搜尋自己有興趣的書籍評論，記錄閱讀心得與評分，並透過推薦排行榜與新書專區發掘更多值得閱讀的書籍。系統頁面設計整潔、易於操作，為讀者打造舒適的閱讀管理體驗。

### 1.3 目標用戶
- **主要用戶：** 讀者（喜歡閱讀並希望記錄讀書心得的一般使用者）
- **使用情境：** 記錄閱讀過的書籍、撰寫心得評分、搜尋書評、瀏覽推薦書單與新書資訊

---

## 2. 功能需求

### 2.1 核心功能清單

| 編號 | 功能名稱         | 優先級 | MVP |
| ---- | ---------------- | ------ | --- |
| F1   | 記錄書名         | 高     | ✅  |
| F2   | 詢問對話框       | 高     | ✅  |
| F3   | 心得與評分       | 高     | ✅  |
| F4   | 推薦書籍排行榜   | 中     | ✅  |
| F5   | 本月新書         | 中     | ✅  |
| F6   | 書籍搜尋         | 高     | ✅  |
| F7   | 書籍分類標籤     | 低     | ❌  |

### 2.2 功能詳細說明

#### F1：記錄書名
- **描述：** 使用者可以新增、編輯、刪除書籍記錄
- **詳細需求：**
  - 新增書籍時需填寫：書名（必填）、作者、出版社、出版日期、ISBN
  - 書籍列表以卡片或表格形式呈現
  - 支援編輯與刪除已記錄的書籍
  - 記錄建立時間與最後更新時間

#### F2：詢問對話框
- **描述：** 提供一個互動式對話框，讓使用者可以針對書籍提出問題或進行討論
- **詳細需求：**
  - 在書籍詳細頁面中設置「詢問 / 留言」對話框區域
  - 使用者可以輸入問題或留言
  - 顯示歷史對話紀錄（依時間排序）
  - 支援回覆功能，形成對話串

#### F3：心得與評分
- **描述：** 使用者可以對已記錄的書籍撰寫閱讀心得並給予評分
- **詳細需求：**
  - 評分採用 1～5 星制
  - 心得為多行文字輸入（支援長篇內容）
  - 每本書可以有多篇心得（不同使用者）
  - 顯示平均評分
  - 心得列表依時間倒序排列

#### F4：推薦書籍排行榜
- **描述：** 根據使用者評分自動產生書籍推薦排行榜
- **詳細需求：**
  - 依平均評分由高至低排序
  - 顯示排名、書名、作者、平均評分、評論數量
  - 預設顯示前 10 名
  - 可依分類篩選排行榜（未來擴充）

#### F5：本月新書
- **描述：** 展示本月新加入系統的書籍
- **詳細需求：**
  - 自動篩選「本月」新增的書籍記錄
  - 以卡片形式展示書名、作者、新增日期
  - 按新增時間倒序排列
  - 支援切換查看不同月份（未來擴充）

#### F6：書籍搜尋
- **描述：** 使用者可透過關鍵字搜尋書籍與書評
- **詳細需求：**
  - 支援以書名、作者進行模糊搜尋
  - 搜尋結果即時顯示
  - 搜尋結果包含書名、作者、平均評分摘要
  - 點擊搜尋結果可跳轉至書籍詳細頁

## 3. 頁面規劃

| 頁面名稱       | 路徑              | 說明                             |
| -------------- | ----------------- | -------------------------------- |
| 首頁           | `/`               | 顯示系統簡介、推薦排行榜、本月新書 |
| 書籍列表       | `/books`          | 所有書籍的列表，支援搜尋與篩選     |
| 新增書籍       | `/books/new`      | 新增書籍表單                      |
| 書籍詳細頁     | `/books/<id>`     | 書籍資訊、心得列表、詢問對話框     |
| 編輯書籍       | `/books/<id>/edit`| 編輯書籍資訊表單                  |
| 推薦排行榜     | `/ranking`        | 書籍評分排行榜                    |
| 本月新書       | `/new-books`      | 本月新增書籍列表                  |

## 4. 非功能需求

### 4.1 使用者介面
- 頁面設計整潔、簡約
- 響應式設計（RWD），支援桌面與行動裝置
- 統一的導覽列（Navbar）方便切換頁面
- 操作回饋明確（成功/失敗提示訊息）

### 4.2 效能
- 頁面載入時間 < 2 秒
- 搜尋回應時間 < 1 秒

### 4.3 資料儲存
- 使用 SQLite 作為資料庫
- 所有書籍與心得資料持久化儲存

## 6. MVP 範圍

**第一版（MVP）包含以下功能：**
1. ✅ 記錄書名（CRUD）
2. ✅ 心得與評分（新增、瀏覽）
3. ✅ 詢問對話框（留言與檢視）
4. ✅ 推薦書籍排行榜（依評分排序）
5. ✅ 本月新書（自動篩選）
6. ✅ 書籍搜尋（模糊搜尋）

**未來擴充：**
- 書籍分類標籤系統
- 使用者帳號與登入功能
- 匯出閱讀清單（CSV / PDF）
- 外部 API 串接（如 Google Books API 自動帶入書籍資訊）
- 閱讀進度追蹤
